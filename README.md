# Better UEFA Gadget

A nostalgic Windows Sidebar Gadget for viewing UEFA football standings and live scores with a retro 2000s web design aesthetic!

## What is this?

This repository contains a fully functional Windows Sidebar Gadget that displays:
- **League Standings** for Eredivisie, Champions League, and Europa League
- **Live Match Scores** with real-time updates and blinking LIVE indicators
- Authentic early-2000s web design with table-based layouts, beveled panels, and classic styling

## Quick Start

1. Get your free API key from [football-data.org](https://www.football-data.org)
2. Check out the `uefa-standings-gadget` folder for the complete gadget
3. Follow the installation instructions in [uefa-standings-gadget/README.md](uefa-standings-gadget/README.md)

## Installation

**Windows Vista/7**: Just create a `.gadget` file and double-click to install!

**Windows 10/11**: Install [8GadgetPack](https://8gadgetpack.net) first to enable gadget support.

Full installation instructions are available in the [gadget README](uefa-standings-gadget/README.md).

- Built with standard HTML, CSS, and JavaScript
- Uses football-data.org API v4
- LocalStorage for settings persistence
- No external dependencies
- Works with Windows Sidebar and 8GadgetPack
